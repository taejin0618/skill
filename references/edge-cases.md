# 엣지케이스 체크리스트

테스트 케이스 작성 시 아래 항목을 80% 이상 커버할 것.

## 입력값 검증

| 카테고리 | 테스트 항목 | 테스트 데이터 예시 |
|----------|------------|-------------------|
| 빈 값 | 공백 제출, 미선택 | `""`, `null`, 체크박스 미선택 |
| 경계값 | 최소-1, 최소, 최대, 최대+1 | 7자/8자/20자/21자 (8~20 제한 시) |
| 특수문자 | SQL Injection, XSS | `' OR '1'='1`, `<script>alert(1)</script>` |
| 유니코드 | 이모지, 한자, RTL 언어 | 😀🎉, 漢字, العربية |
| 공백 | 앞뒤/중간 공백, 개행 | `"  text  "`, `"a    b"`, `"line1\nline2"` |
| 대소문자 | 전체 대문자/소문자/혼합 | `HELLO`, `hello`, `HeLLo` |

## 동작/상태 검증

| 카테고리 | 테스트 항목 | 확인 사항 |
|----------|------------|----------|
| 중복 요청 | 더블클릭, 동시 호출 | 중복 처리 방지 확인 |
| 뒤로가기 | 브라우저 뒤로가기 후 재요청 | 중복 제출/데이터 손실 확인 |
| 타임아웃 | 느린 네트워크, 끊김 | 에러 핸들링, 재시도 로직 |
| 세션 | 만료, 동시 로그인 | 적절한 에러 메시지 |

## 권한/보안 검증

| 카테고리 | 테스트 항목 | 확인 사항 |
|----------|------------|----------|
| 비로그인 | 인증 필요 페이지 접근 | 로그인 페이지 리다이렉트 |
| 권한 부족 | 관리자 기능 일반 사용자 접근 | 403 또는 적절한 에러 |
| 타인 데이터 | URL 파라미터 조작 | 접근 차단 확인 |

## 환경/호환성 검증

| 카테고리 | 테스트 항목 | 확인 사항 |
|----------|------------|----------|
| 브라우저 | Chrome, Safari, Firefox, Edge | 동일 동작 확인 |
| 디바이스 | 모바일, 태블릿, 데스크톱 | 반응형 UI 확인 |
| 해상도 | 360px ~ 1920px+ | 레이아웃 깨짐 없음 |
