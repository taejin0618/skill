## 필수 체크 항목

### 1. 기능 커버리지 (100% 필수)

화면에 보이는 모든 기능에 대해 테스트 케이스가 존재하는지 확인

| 체크 항목 | 확인 |
|----------|------|
| 모든 버튼에 대한 클릭 테스트 | ☐ |
| 모든 입력란에 대한 입력 테스트 | ☐ |
| 모든 드롭다운/체크박스/라디오 버튼 테스트 | ☐ |
| 페이지네이션 테스트 | ☐ |
| 검색/필터 기능 테스트 | ☐ |
| 정렬 기능 테스트 | ☐ |
| 다운로드/업로드 기능 테스트 | ☐ |
| 출력/인쇄 기능 테스트 | ☐ |

### 2. 엣지케이스 커버리지 (80% 이상 권장)

| 체크 항목 | 확인 |
|----------|------|
| 빈 값 / null 입력 | ☐ |
| 최소값 / 최대값 (경계값) | ☐ |
| 특수문자 / 이모지 / 유니코드 | ☐ |
| SQL Injection 패턴 | ☐ |
| XSS 공격 패턴 | ☐ |
| 앞뒤 공백 / 중간 공백 | ☐ |
| 더블클릭 / 중복 요청 | ☐ |
| 브라우저 뒤로가기 | ☐ |
| 세션 만료 | ☐ |
| 네트워크 타임아웃 | ☐ |

### 3. 보안 커버리지 (필수)

| 체크 항목 | 확인 |
|----------|------|
| 비로그인 사용자 접근 차단 | ☐ |
| 권한 없는 사용자 접근 차단 | ☐ |
| 타인 데이터 접근 차단 (URL 조작) | ☐ |
| 타 조직 데이터 접근 차단 | ☐ |
| SQL Injection 방어 | ☐ |
| XSS 공격 방어 | ☐ |

### 4. UX 커버리지 (권장)

| 체크 항목 | 확인 |
|----------|------|
| 로딩 인디케이터 표시 | ☐ |
| 성공/실패 피드백 메시지 | ☐ |
| 에러 메시지에 해결 방법 포함 | ☐ |
| 삭제 등 중요 작업 확인 다이얼로그 | ☐ |
| 모달/팝업 닫기 버튼 | ☐ |
| 취소/되돌리기 기능 | ☐ |
